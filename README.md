# Geo-Mind-game
A basic game based on user guessing country's name based on some clues. It has points system with 10 rounds, each extra clue needed will deduct some points. 
This is a fast-paced country guessing game built using "Vibe Coding" principles.

# The Vibe
This project represents my first step into AI-orchestrated development. Instead of manual line-by-line coding, I focused on the "logic and user experience", using Claude to generate the functional core.

# Features
- "Point-Based Scoring": Track your geographic knowledge in real-time.
- "Instant Feedback": Learn about each country as you play through the clues.
- "Clean UI": Minimalist design focused on speed.

# Tech Stack
- HTML5 / CSS3 / JavaScript
- Architected via "Claude AI" (Vibe Coding)
